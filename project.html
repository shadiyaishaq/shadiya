Recent Projects

Galeto : A Seamless Mobile Experience for Galeto Ice Cream Delivery

Galeto is a user-friendly mobile application designed to simplify the process of ordering custom ice cream creations and popular flavors for delivery. The core goal of this project was to leverage clean, high-contrast UI design and an intuitive information architecture to transform browsing into a quick, enjoyable ordering experience. We focused on making product discovery effortless, ensuring that users can easily explore flavors, find personalized offers, and access top-rated items with minimal taps. This design demonstrates expertise in mobile-first UX principles, clear hierarchy, and optimizing the checkout flow for speed.

Designing for Tomorrow : A Tech-Savvy and Clean E-Commerce Platform for EcoDrive

EcoDrive is a startup selling premium, eco-friendly electric scooters, and this project focused on designing a highly modern and tech-savvy brand website to drive initial adoption and sales. The primary challenge was to showcase complex scooter models and specifications within a simple, engaging flow. We achieved this by utilizing a futuristic typography and a clean, dark-mode-inspired UI to establish a high-end, contemporary feel. The design features clean, modular sections and high-quality product photography to ensure the scooter models are the central focus. This case study demonstrates mastery in visual storytelling, responsive web design, and creating an immersive experience that builds trust in a new electric brand.

RideMate: Designing a Seamless, Real-Time Mobile Experience for Urban Bicycle Rentals

RideMate is a bicycle rental application designed to provide users with a fast, reliable way to locate, view, and unlock bikes across the city. The core design challenge was creating a clean, real-time layout that balances essential map data with key bike details, ensuring users can transition from browsing to riding in minimal time. We implemented an eco-theme through color and typography while prioritizing the map view as the main interface for discovery. This project demonstrates mastery in real-time UI design, streamlined transactional flows, and optimizing mobile interfaces for on-the-go usability.

MindScape: Bridging Healing Modalities with a Calm, Accessible Digital Platform

MindScape is a digital wellness platform (mobile app and companion website) meticulously designed to support individuals navigating mental health challenges. The core objective was to create an empathetic, non-triggering, and deeply supportive UI/UX that facilitates healing through various therapeutic modalities like guided yoga, reflective journaling, and engaging tasks. My design approach prioritized calmness, clarity, and accessibility, ensuring a safe digital space that promotes user agency and self-care. This case study demonstrates expertise in designing for sensitive contexts, robust information architecture for therapeutic content, and building a unified, compassionate user experience across platforms.

CargoWave: Optimizing Global Logistics Through Intuitive Data Visualization

CargoWave is a comprehensive B2B platform designed to give businesses real-time visibility and predictive analytics for complex global logistics. The core challenge was transforming fragmented and clunky industry data into a unified, intuitive dashboard. I led the effort to create a modern web application that not only reduces operational errors but empowers users to make fast, data-driven decisions. This case study demonstrates expertise in complex information architecture, clean data visualization UI, and creating scalable, high-performance interfaces built for speed and efficiency.
